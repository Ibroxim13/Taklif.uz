# 💍 Nilufar Studio — To'y Taklifnomalar Sayti

Mobil-birinchi, to'liq responsive shaxsiy brend sayti.

## 🚀 Ishga tushirish

```bash
npm install
npm run dev
```

Sayt `http://localhost:3000` da ochiladi.

## 🛠 Texnologiyalar

| Texnologiya | Maqsad |
|-------------|--------|
| React 18 | UI framework |
| Vite | Build tool |
| Tailwind CSS | Utility styling |
| Framer Motion | Animatsiyalar |
| React Intersection Observer | Scroll animatsiyalar |
| React Icons | Ikonalar |
| React Hot Toast | Xabarnomalar |

## ✏️ Sozlash

**Telegram:** `src/components/Contact.jsx` va `Products.jsx` da  
`nilufar_studio` → o'zingizning username'ingiz

**Rang sxemasi:** `src/index.css` da CSS o'zgaruvchilari:
```css
:root {
  --rose: #C9547C;
  --gold: #C9A84C;
  --cream: #FAF5EE;
}
```

**Mahsulotlar:** `src/data/index.js` faylini tahrirlang
